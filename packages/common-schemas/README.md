# Common Schemas

This package holds shared contracts for Eidonic Core services.

## Current scope
The first shared Python package provides common Pydantic request models for:
- signal-gateway
- herald-service
- session-engine
- eidon-orchestrator

## Purpose
These models reduce contract drift between services and create a single source of truth for core request payloads during Phase 1.

## Local install
Service requirements install this package in editable mode from:

`../../packages/common-schemas/python`
